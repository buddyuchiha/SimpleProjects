# CurrencyExchange REST API

REST API для управления валютами и курсами обмена с функциональностью конвертации

[Техническое задание](https://github.com/user-attachments/files/23303680/tech.1.pdf) 

## Функционал

* Управление валютами: получение списка, данных по коду, добавление новых
* Управление курсами обмена: получение списка, данных по паре, добавление и обновление
* Конвертация валют с поддержкой трех сценариев получения курса
* Обработка HTTP-запросов с соответствующими кодами ответов
