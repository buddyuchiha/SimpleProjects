# Hangman Game

Классическая консольная игра "Виселица" с угадыванием слов по буквам

## Функционал

* Угадывание случайных слов из словаря по одной букве
* Визуализация прогресса через ASCII-рисунок виселицы
* Отслеживание использованных букв и оставшихся попыток
* Возможность перезапуска игры и смены загаданного слова
* Игнорирование регистра букв (А/а считаются одинаково)
